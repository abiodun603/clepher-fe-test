## Installation
yarn install

## Usage
yarn dev

## Set UP
 - Add .env.local file
 - Add the API Key, name it NEXT_PUBLIC_STOCKS_API_KEY
 - Add the API base url, name it NEXT_PUBLIC_STOCKS_API_URL


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
